# Read Knowledge Tree

Central repository for reading learning across all readers (Igor, future readers, etc.).

## Structure

- `book_learner_progress/` — Book+position learning progress (which positions in each book have been processed)
- `reading_state.json` — Current reading state (position, progress % per book)

## Ownership

Owned by **Librarian**. All readers reference this tree to build their personal understanding weights and connections.

## Readers

- Igor Wild0 (archived) — original learning data source
- Igor Wild1 — fresh instance, builds new weights on this tree
- Future readers — all contribute to and learn from this shared knowledge base
