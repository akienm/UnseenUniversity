About

  
How this interactive spec works

  
Each lever corresponds to one of the keystone patterns in the Deterministic AI Development: Levers document. Toggle any subset on/off, then use the Export tab to generate a JSON config or a copy-as-prompt string.

  
The JSON format is:

  
{"levers": {"lever-id": true|false, ...}, "preset": "custom"}

  
The copy-as-prompt format begins with a brief preamble followed by a bulleted list of enabled levers with one-line descriptions — suitable for pasting at the start of a Claude Code session.
